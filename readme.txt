Udimi Optin Tracking simplifies process of adding optin tracking script from Udimi.com to your site

== Description ==

This plugin inserts an optin tracking javascript code to your site's head and keeps it up-to-date by loading updates (if any) from Udimi.com site every 24 hours.

== Installation ==

1. Activate the plugin

2. Log into your Udimi.com account and click "Swipes" link in the left menu

3. Press "New Swipe" button or click on any swipe in the list

4. Fill "Subject" and "Text" fields and click "Save" button

5. Enable optin tracking and check checkbox "My site is powered by WordPress"

6. Make sure you are logged in to your Udimi account, go to plugin's settings page and press "Connect" button
